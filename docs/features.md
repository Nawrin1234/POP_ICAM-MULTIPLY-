# Current Features

## 1. Dynamic Math Engine
- **Difficulty Scaling**: Three modes (Easy, Medium, Hard) that adjust the range of target numbers and the frequency/value of bubbles.
- **Real-time Summation**: Tracks current player sum with visual feedback for "going over" the target.
- **Timer Mechanics**: A countdown bar adds urgency, with bonus points awarded for speed.

## 2. Interactive UI/UX
- **Bubble Physics**: Bubbles float and bounce with randomized speeds and sinusoidal movement to keep the game visually engaging.
- **Feedback Systems**: 
  - **Audio**: Custom synthesized sounds for popping (success), errors (exceeding sum), and level completion.
  - **Visuals**: A "Magic Meter" that pulses and glows as it fills up.
  - **Pop Animation**: Satisfying burst effect when bubbles are tapped.

## 3. The Creator Forge (AI Rewards)
- **Gemini 3 Pro Integration**: Uses the latest Image Generation models to create Pixar-style, child-friendly illustrations.
- **Customizable Prompts**: Children can choose from presets (e.g., "A friendly dragon eating chocolate cupcakes") or type their own imagination.
- **Configurable Output**: High-quality options for aspect ratios (1:1 to 16:9) and resolutions (1K to 4K).

## 4. Persistence
- **High Scores**: Saves personal bests for each difficulty level to the browser's local storage.
- **API Safety**: Integrated API key selection logic to ensure high-quality generation is accessible only with valid credentials.
